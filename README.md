# JumeTo 🍽️

**Discover food you love** — a modern, responsive food discovery and ordering web application.

## Overview

JumeTo is a single-page web application that lets users browse local restaurants, explore menus, add items to a cart, and place orders — all from a beautifully designed interface with full dark mode support.

## Features

- 🔍 **Search** — Find dishes or restaurants instantly from the hero search bar or the top navigation
- 📍 **Location-based browsing** — Browse restaurants by city/area
- 🏠 **Home feed** — Curated sections for trending restaurants, top-rated picks, and popular cuisines
- 🍴 **Restaurant listing** — Filter and sort restaurants by cuisine, rating, and delivery time
- 📋 **Menu view** — Browse categorised menus with item details, ratings, and add-to-cart controls
- 🛒 **Cart & Checkout** — Manage your order, apply offers, and complete checkout
- 👤 **Authentication** — Sign in / sign up with phone, email, or social providers
- ⚙️ **User Profile** — View order history, saved addresses, and manage account settings
- 🌙 **Dark Mode** — Toggle between light and dark themes
- 📱 **Mobile-first design** — Fully responsive layout with a dedicated mobile bottom navigation bar

## Getting Started

No build step or server required — JumeTo runs entirely in the browser.

1. **Clone the repository**

   ```bash
   git clone https://github.com/anmolsin1124/jumeto.git
   cd jumeto
   ```

2. **Open in your browser**

   ```bash
   open index.html
   ```

   Or simply drag `index.html` into any modern web browser.
